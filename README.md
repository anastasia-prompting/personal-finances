# Личный финансовый AI-аудитор — Android MVP

Нативный Android-проект на Java/Kotlin-compatible Android SDK без серверной части.

## Что реализовано в v1

- Нативный Android-проект с `MainActivity` и локальной SQLite-базой.
- Две темы интерфейса: светлая и тёмная.
- Пользовательское соглашение при первом запуске и повторный просмотр из главного экрана.
- Быстрый ввод операций:
  - Доход
  - Расход
  - Перевод в фонд
  - Расход из фонда
  - Платёж по кредиту
  - Увеличение долга
- Счета: основная карта, наличные, кредитная карта.
- Фонды: подушка, жильё, развитие, удовольствия, Ozon, Wildberries.
- Категории доходов и расходов.
- Кредитная логика: долг растёт через «Увеличение долга» и уменьшается через «Платёж по кредиту».
- Главный экран с балансами, фондами, долгом, доходами и расходами.
- Журнал последних операций.
- Удаление операций.
- Экспорт CSV в app-specific external storage.

## Как открыть

Открыть папку `FinancialAIAuditor` в Android Studio.

## Как собрать APK

В Android Studio:

1. File → Open → выбрать папку `FinancialAIAuditor`.
2. Дождаться Gradle Sync.
3. Build → Build Bundle(s) / APK(s) → Build APK(s).
4. APK будет в `app/build/outputs/apk/debug/app-debug.apk`.

Или через терминал, если Gradle Wrapper создан Android Studio:

```bash
./gradlew assembleDebug
```

## Важно

В этой среде сборки нет Android SDK / Gradle, поэтому APK здесь не собран. Проект подготовлен для сборки в Android Studio.
