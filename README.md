# Личный финансовый AI-аудитор (MVP)

React + TypeScript + Vite + Tailwind + **Dexie (IndexedDB)** + PWA, mobile-first. Без бэкенда, без банков и без облачной синхронизации.

## Скрипты

- `npm run dev` — разработка
- `npm run build` — production
- `npm test` — Vitest (базовые сценарии)

## Капаситор (позже)

Сборка: `npm run build`, затем `npx cap add android` в отдельной ветке; точка входа — `dist/`.

## Принцип

Ясность без давления, управление без жёсткости, учёт без перегруза.
